# 🤖 Sidet

Obsidian 的 AI 工作区插件。将对话放在主工作区，支持带入笔记、选区、网页、图片等材料，预览确认后写回笔记。

---

## 📌 用途

Sidet 把 AI 对话放在 Obsidian 的主工作区，而非侧边窄栏。

你在完整的编辑区域里与 AI 沟通，直接把当前笔记、选区、网页、图片等材料带入上下文。聊完先预览，确认后再决定写回方式: 创建新笔记、追加现有笔记，或改写目标笔记。AI 不会未经同意就写入你的库。

对话历史支持搜索、置顶和文件夹分类。手机端独立优化，不是桌面版的简单缩放。

适合长期在 Obsidian 中读写整理、希望 AI 贴着工作流发生的人。不需要切到另一套 AI 软件，也不需要手动搬运材料。

> [!NOTE]
> 如果你需要重型 agent 平台、满屏参数或后台自动化，Sidet 不是按那条路线做的。

---

## ⚙️ 工作流程

1. 打开 Sidet 标签页，新建标签页直接呈现对话框
2. 将当前笔记、选区、网页、图片等材料带入对话
3. 与 AI 沟通，边聊边整理
4. 预览 AI 输出，确认后写回笔记库

---

## ✨ 功能

| 功能 | 说明 |
|:---|:---|
| 💬 主工作区对话 | 当前对话占用主标签页，侧栏仅用于会话列表 |
| 📥 上下文带入 | 当前笔记、选区、相关笔记、文件夹摘要、网页、图片 |
| 📝 可控写回 | 预览后选择: 创建新笔记、追加现有笔记、或改写目标笔记 |
| 📂 会话管理 | 侧栏支持搜索、置顶、文件夹整理、历史保留、导出 |
| 🏠 命令主页 | 新建标签页直接呈现对话框，主页固定常用命令 |
| 📚 知识库沉淀 | 对话或笔记整理到知识库，为文件夹生成 Index.md |
| 📱 移动端适配 | 针对手机端独立优化布局与交互 |

---

## 📝 写回机制

不提供默认自动写回。AI 输出需经预览确认，再选择入库方式:

- 创建新的 Markdown 笔记
- 追加到现有笔记末尾
- 改写目标笔记内容

---

## 📚 知识库

Sidet 以 AI 对话为主轴。聊到有价值内容时，可选择性沉淀:

- 将当前对话整理进知识库
- 将当前笔记整理进知识库
- 为文件夹创建 Index.md 作为知识库入口

## 🚀 安装

> [!WARNING]
> Sidet 不在官方 Community Plugins 目录中，应用内无法直接搜索安装。

### 方式一: BRAT（推荐）

1. 在 Obsidian 社区插件中安装 **BRAT**
2. 打开 BRAT，选择 **Add Beta plugin**
3. 输入 `https://github.com/jiaoyingxing/sidet`
4. 安装完成后，在 Obsidian 设置中启用 **Sidet**

### 方式二: 手动安装

1. 从 [GitHub Releases](https://github.com/jiaoyingxing/sidet/releases) 下载安装文件
2. 放入 vault 的 `.obsidian/plugins/sidet/` 目录
3. 重启 Obsidian 或重新加载社区插件
4. 在设置中启用 Sidet
