# Sidet

Sidet is a lightweight Obsidian chat workspace with side browsing, focused threads, and note-aware knowledge capture.

This public repository is the release-artifact repository for Sidet.

It intentionally contains only the files needed for installation:

- `main.js`
- `manifest.json`
- `styles.css`
- `versions.json`

Source code, development documents, scripts, and local runtime data stay in the private source repository.

## Install

Download the release files for the target version and place them in:

```text
.obsidian/plugins/sidet/
```

Required files:

- `main.js`
- `manifest.json`
- `styles.css`

## Current Positioning

- Side browsing for conversations
- Focused main chat view
- Hidden conversation storage with on-demand export
- Note-aware knowledge capture inside Obsidian

## Notes

- This repository is for public artifacts only.
- No source code is published here.
